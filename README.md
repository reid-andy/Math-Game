# Math Game
---
A simple math game developed following The C# Academy's Roadmap.
https://thecsharpacademy.com/project/53/math-game

## How to Play
Run the program in a console window. Input a number from the menu list to select a gametype and difficulty or to see the scoreboard. 

Complete the selected math operations as shown. Your correct answers will be logged as your score as well as your completion time in seconds. 

Have fun!
